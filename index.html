Initiation à HTML & CSS

Objectifs

  • Comprendre le fonctionnement d'une page web
  • Comprendre la distinction entre HTML et CSS
  • Réaliser sa première page web

Différence entre HTML et CSS

HTML

HTML est l'acronyme de HyperText Markup Language. Une page HTML est ce qui sous-tend le web : c'est ce format qui sera lu par votre navigateur web. Une page HTML est constituée :

  • du contenu de la page, c'est-à-dire le texte à afficher sur l'écran
  • d'un ensemble de balises pour indiquer le "rôle" de chaque élément texte. Par exemple, si cette phrase est un titre, une citation, du corps de texte, un sous-titre, une note de bas de page, etc.
CSS

CSS est l'acronyme de Cascading Style Sheets. Ces fichiers décrivent la présentation d'une page, c'est-à-dire sous quelle forme le contenu sera affiché. Par exemple, une feuille CSS peut décrire pour une page :

  • un titre doit apparaître de manière centrée, en gras, souligné avec une police Arial
  • un paragraphe doit apparaître aligné à gauche, en bleu foncé avec la police Times
